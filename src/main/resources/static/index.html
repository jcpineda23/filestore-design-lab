Failure Drill Guide

Use this while testing C3 and C4. Stop MinIO from the terminal, then watch health, file states, and SSE events update here.

  1. Connect the event stream.
  2. Click Snapshot State to capture a healthy baseline.
  3. Stop MinIO in the terminal with docker compose stop minio.
  4. Click Note MinIO Stopped, then try an upload.
  5. Watch for STORAGE_UNAVAILABLE, a file state of FAILED, and a file.upload.failed event.
  6. Start MinIO again with docker compose up -d minio and click Note MinIO Restored.
